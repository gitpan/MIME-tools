NAME

mimeref - create a .ref file for a message

SYNOPSIS

Usage: 

    mimeref [-options] *.msg

Options: 

    -d DIR    Output directory for parser (default is ./testout/mimeref)
    -w        Write the .ref file

DESCRIPTION

Parse a message file, and spit out a .ref file. The .ref files are not really useful; they're just used by the t/Ref.t test.
