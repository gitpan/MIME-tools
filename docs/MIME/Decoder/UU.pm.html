NAME

MIME::Decoder::UU - decode a "uuencoded" stream

SYNOPSIS

A generic decoder object; see MIME::Decoder for usage.

DESCRIPTION

A MIME::Decoder subclass for a nonstandard encoding whereby data are uuencoded. Common non-standard MIME encodings for this: 

    x-uu
    x-uuencode
